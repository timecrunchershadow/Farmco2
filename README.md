# FarmTrak - Strategic Intelligence Hub

Welcome to FarmTrak, a strategic intelligence application built to provide Farmco with a competitive edge in the agricultural equipment market. This tool allows for in-depth competitor analysis, product comparison, and centralized data management, all powered by AI-driven insights.

## Key Features

- **Competitor Battlecards**: Create and manage detailed profiles for each competitor. Track their strengths, weaknesses, closing narratives, and field objections in one place.
- **AI-Powered Summary**: Leverage AI to automatically generate concise summaries of a competitor's business directly from their website URL.
- **Dynamic Spec Comparison**: Build side-by-side comparisons of Farmco products against competitor equipment. Autopopulate Farmco specs from a master database.
- **Master Product Database**: Maintain a centralized database of all Farmco products, including specifications, variants, and pricing.
- **Image Gallery**: Upload and manage galleries of competitor product photos for visual analysis.

## Tech Stack

- **Framework**: Next.js (App Router) & React
- **Database**: Google Firestore
- **Styling**: Tailwind CSS with shadcn/ui components
- **Generative AI**: Google Gemini via API calls
- **Deployment**: Firebase App Hosting

## Getting Started

This application is designed to run in a cloud-based development environment like Firebase Studio.

To start the development server, run the following command:

```bash
npm run dev
```

This will start the Next.js development server, and you can access the application in your browser. All data is stored in a live Firebase project, so any changes you make will be persistent.
